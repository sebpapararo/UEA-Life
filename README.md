  # UEA Life ReadMe (DSS Group Coursework)

  UEA Life is a python flask website designed to allow students and or staff to
  post comments and replies in the form of a basic forum.

  ## Installation

  ```bash
  pip install flask
  pip install flask-mail
  ```

  ## Usage
  
  ```bash
  python ./app.py
  ```
